# Academic Portfolio - Samrat Kumar Dev Sharma

This is a highly customizable, responsive academic portfolio website built with pure HTML, CSS, and JavaScript. It features a clean, modern design tailored for researchers, showcasing publications, projects, and academic achievements.

## 🚀 Features
- **Responsive Design:** Looks perfect on mobile, tablet, and desktop.
- **Dark/Light Mode:** Built-in theme toggle with local storage memory.
- **Dynamic Abstracts:** Clickable paper titles that load abstracts dynamically without page reloads.
- **Zero Dependencies:** No heavy frameworks (like React or Hugo). Just pure, fast code.

## 📂 Project Structure
- `index.html`: Main landing page.
- `publications.html`: Complete list of papers and projects.
- `paper-details.html`: Dynamic page for viewing paper abstracts.
- `css/style.css`: All styling and CSS variables for easy theming.
- `js/script.js`: Theme toggling, smooth scrolling, and dynamic data loading.

## 🛠️ How to Edit
1. Open `index.html` to change your personal info and profile.
2. Open `paper-details.html` and edit the `papersDatabase` JavaScript object to add your own papers and abstracts.
3. Place your PDF files in the `assets/pdf/` folder.
